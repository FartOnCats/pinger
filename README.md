# pinger
A tool to test connection to a website<br/>
To use the file %appdata%/ping.txt must be present<br/>
the format for the txt file doesnt matter much aslong as each address starts with a colon(:)<br/>
example<br/>
ip:192.168.0.1<br/>
ip:google.com<br/>
ip:192.168.0.144<br/>
ip:youtube.com<br/>
